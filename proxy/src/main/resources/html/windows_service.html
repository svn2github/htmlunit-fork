Windows Service

Installation

To install PAW as a Windows Service double-click the file windows-service\InstallApp-NT.bat in the PAW installation directory. After the Installation the PAW service should be available as a Windows Service.


Removal

To uninstall PAW Windows Service double-click the file windows-service\UninstallApp-NT.bat in the PAW installation directory.



 Note

On Windows Vista the bat files have to be run as Administrator!
