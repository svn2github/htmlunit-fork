Admin-Server
To configure the PAW AdminServer, simply select the node in the tree-view. The Admin-Server Configuration will be displayed in the right window.

Here you can change the port, user and password for the Admin Server or if you want to use an Admin Server at all.

Note:
If you deselect the Use Admin Server checkbox (and save the configuration you will not be able to manage this PAW Server via the PAW GUI.

After you have finished press in the configuration window. To save back the whole configuration use the save button in the toolbar.
